# Material Usage — Ombor Materials Per Step

## What it does

Tracks which ombor materials were consumed at each production step. This is how you calculate cost per step and total order cost.

## Existing model (with new `step_id` field)

```
MaterialUsage:
  ombor_transaction_id  → linked chiqim transaction
  brigada_id            → which brigada
  machine_id            → which stanok
  plan_item_id          → which order
  ombor_item_id         → which material (plyonka, kraska, kley, etc)
  step_id               → NEW: which step consumed it
  issued_amount         → how much taken from ombor
  used_amount           → how much actually used
  remainder             → leftover
  remainder_destination → went back to ombor or stayed at machine
```

## Typical materials per step

| Step | Materials from ombor |
|------|---------------------|
| Reska (initial) | Plyonka (if first step) |
| Pechat | Kraska |
| Sushka | None |
| Laminatsiya | Kley + plyonka (extra layer) |
| Reska (final) | None |
| Tayyor | None |

### List material usage for a specific step

`GET /plan-items/{id}/steps/{step_id}/material-usage`

**Response (200)** — same format, filtered by step.

## Cost calculation

Cost is NOT stored. It's calculated by joining MaterialUsage with OmborItem prices.

### Per step cost

```
Step 2 (Pechat):
  MaterialUsage: used_amount=28kg, ombor_item_id=45
  OmborItem #45: price_per_kg=8.0, price_currency=usd

  Cost = 28 * $8 = $224
```

### Total order cost

```
SUM of all MaterialUsage for plan_item_id=10:
  Step 1: 490kg plyonka × $2.5  = $1,225
  Step 2: 28kg kraska × $8      = $224
  Step 4: 8kg kley × $12        = $96
  Step 4: 80kg plyonka × $2.5   = $200
  ─────────────────────────────────
  Total: $1,745
  Final product: 460kg
  Cost per kg: $3.79/kg
```
